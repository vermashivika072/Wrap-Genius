# 🎁 Wrap Genius

Gift Petal

A beautifully designed gift website that helps users explore, customize, and send perfect gifts for every occasion.

---

## 🌸 About the Project

**Wrap Genius Gift Petal** is an online gifting platform where users can browse a variety of gift items, personalize them, and send them to loved ones with ease.

The platform focuses on:
- Elegant UI/UX 🎨
- Smooth user experience ⚡
- Smart gift selection 💡

---

## 🚀 Features

- 🎁 Browse a wide range of gifts
- 🛒 Add to cart functionality
- 💳 Easy checkout process
- 🎨 Beautiful and responsive design
- 🔍 Search and filter gifts
- ❤️ Personalized gifting options
